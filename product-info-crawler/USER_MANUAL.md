# 상품 정보 크롤러 사용 방법

## 1. 처음 설치

1. 프로젝트 폴더를 엽니다.
2. `설치.bat`를 실행합니다.
3. 설치가 끝날 때까지 기다립니다.

## 2. 실행 전 준비

- `credentials/service_account.json` 파일이 있어야 합니다.
- Google Sheets 저장을 쓰려면 대상 시트가 서비스 계정과 공유되어 있어야 합니다.
- 가져올 카테고리 URL은 `config` 폴더의 브랜드 설정 파일에 미리 입력되어 있어야 합니다.

## 3. 프로그램 실행

1. `실행.bat`를 실행합니다.
2. 브라우저에서 열린 화면에서 브랜드를 선택합니다.
3. 저장 대상을 선택합니다.
4. 저장 방식을 선택합니다.
5. 수집할 카테고리를 선택합니다.
6. `크롤링 시작` 버튼을 누릅니다.

## 4. 저장 대상

- `Google Sheets`: Google Sheets에 저장합니다.
- `Excel`: 바탕화면의 `Brand_Product_Data` 폴더에 브랜드별 엑셀 파일로 저장합니다.

## 5. 저장 방식

- `덮어쓰기`: 같은 시트가 있으면 기존 내용을 지우고 다시 저장합니다.
- `이어쓰기`: 같은 시트가 있으면 기존 내용 아래에 계속 추가합니다.

## 6. Excel 저장 위치

- 저장 폴더: 바탕화면 `Brand_Product_Data`
- 파일명: 선택한 브랜드명과 동일
- 시트명: 선택한 카테고리명

## 7. 실행 중 확인할 수 있는 내용

- 현재 처리 중인 카테고리
- 전체 진행률
- 카테고리별 성공/실패 결과
- 카테고리별 저장 건수

## 8. 주의 사항

- 실행 중 브라우저를 임의로 닫지 않는 것이 좋습니다.
- 브랜드 사이트 구조가 바뀌면 수집이 실패할 수 있습니다.
- 실패한 카테고리가 있어도 나머지 카테고리는 계속 진행됩니다.

## 9. 문제 발생 시 확인

- 서비스 계정 파일 경로가 맞는지 확인합니다.
- 대상 Google Sheets 이름과 공유 권한을 확인합니다.
- Excel 저장 시 바탕화면에 `Brand_Product_Data` 폴더가 생성됐는지 확인합니다.
- `config` 파일의 카테고리 URL이 올바른지 확인합니다.
- 화면에 표시된 실패 메시지를 확인합니다.
