\[쇼핑몰 크롤러 사용법]



1\. 최초 1회 설치

&nbsp;  1) shopping-crawler 압축을 풀어주세요.

&nbsp;  2) 폴더 안에 있는 "설치.bat" 를 더블클릭합니다.

&nbsp;  3) 까만 창에서 설치가 끝날 때까지 기다렸다가, "아무 키나 누르세요..." 가 나오면 창을 닫습니다.



2\. 프로그램 실행

&nbsp;  1) 사용할 때마다 "실행.bat" 를 더블클릭합니다.

&nbsp;  2) 잠시 후 브라우저(크롬 등)가 자동으로 열리고, 화면에서

&nbsp;     - 브랜드 선택 (Celine / Bottega)

&nbsp;     - 크롤링 시작 버튼 클릭

&nbsp;     순서로 진행하시면 됩니다.



3\. 결과 확인

&nbsp;  - 결과는 미리 설정해 둔 구글 시트에 자동으로 저장됩니다.

&nbsp;  - 시트 이름과 위치는 미리 안내드린 링크에서 확인하실 수 있습니다.



※ 주의사항

\- 크롬 브라우저가 설치되어 있어야 합니다.

\- 프로그램 실행 중에는 브라우저를 마음대로 닫지 말아 주세요.

\- 문제가 생기면 화면에 나온 오류 메시지를 캡쳐해서 보내 주세요.
